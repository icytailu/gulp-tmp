# 必读

> 项目基础目录

```
|-- src                   // 源文件
|   |-- css               // css文件
|   |--  |-- index.styl   // 引入所有依赖样式
|   |--  |-- common.styl  // 基础样式
|   |-- imgs              // 图片
|   |-- js                // 脚本文件
|   |-- views             // 静态页面
|-- index.html            // 项目首页
```

## 开始

```npm

npm i

npm run dev

```

## 打包

```npm

npm run build

```
